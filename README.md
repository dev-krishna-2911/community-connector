# CampusConnect

## About the Project

CampusConnect is a backend application built with **Node.js, Express.js, and MongoDB** that serves as a centralized platform for college communities. It enables students to discover, join, and actively participate in various campus communities based on their interests.

The platform provides secure authentication and user management, allowing students to collaborate, participate in community events, and contribute to meaningful activities. To encourage engagement, members earn **reward points** for their contributions, helping them increase their **level** and showcase their progress within the platform.

CampusConnect also features a **ranking system** where each community member is evaluated based on their contributions. Rankings are assigned by the **Community Leader**, who is democratically elected through a voting process by the members of that community. This creates a transparent and community-driven leadership model while motivating members to stay active and contribute consistently.

The goal of CampusConnect is to build an engaging ecosystem where students can learn, collaborate, gain recognition for their efforts, and grow both personally and professionally through active community participation.

## Installation

Clone the repository:

```bash
git clone https://github.com/dev-krishna-2911/community-connector.git
```

Move into the project folder:

```bash
cd community-connector
```

Install dependencies:

```bash
npm install
```

## Environment Variables

Create a `.env` file in the project root.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## Running the Project

Development mode:

```bash
npm run dev
```

Production mode:

```bash
npm start
```
